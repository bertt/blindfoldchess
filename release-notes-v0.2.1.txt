### Installation
Download the appropriate executable for your platform and run it. No installation required!

**Windows:** `blindfoldchess.exe`  
**Linux/macOS:** `blindfoldchess` (requires .NET 8.0)

Or build from source:
```bash
dotnet build
cd src/Chess.Console
dotnet run
```
